## Prerequisites

- npm >=5.5.0
- node >=9.3.0

## Install

```sh
nvm install && nvm use
npm install
```

## Create db 

```sh
createdb (name database)
```

## Drop data base in console 

```sh
dropdb (name database)
```

## Run migrations in console

```sh
nmp run db:migrate
```

## Run seeds

```sh
nmp run db:seed
```

## Usage

```sh
npm run dev
```

## Run tests

```sh
npm run test
```
